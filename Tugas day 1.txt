Summary

Hari ini belajar mengenai apakah yang dimaksud dengan back end developer, ruang lingkup back end, mengenal beberapa istilah yang sering muncul ketika akan
melakukan pemrograman serta beberapa tools yang akan digunakan ke depan.

back end adalah bagian dari lingkup developer yang memfokuskan diri pada pengolahan data server beserta API yang nantinya akan diteruskan
ke bagian front end (client side).

Analogi yang cocok untuk back end yang sudah terkenal adalah seperti dalam sebuah restoran, maka back end adalah koki dan pelayannya. yang dimana koki dan pelayan
akan memproses sebuah masakan (data) yang nantinya akan diteruskan ke customer (client side)

API adalah penerjemah aplikasi antar klien seperti contohnya ketika membuka dari web browser, android maupun IOS akan memiliki tampilan dan hasil yang sama
dari programming yang telah dibuat sebelumnya, dalam artian tersingkronisasi mau buka dari device manapun.

Terminal adalah sebuah piranti yang digunakan untuk berkomunikasi dengan komputer, dengan menginputkan beberapa perintah tertentu, maka si komputer akan
dapat menampilkan sesuai input yang dimaksud.

beberapa tools yang akan digunakan kedepannya adalah visual studio code, node JS dan Git bash.
vs code nantinya akan digunakan untuk menuliskan baris-baris pemrograman
node JS digunakan untuk mengembangkan aplikasi berbasis web menggunakan javascript yang dapat berjalan di sisi server (back end)
Git bash dapat digunakan sebagai changelog dari sebuah baris pemrograman sehingga para developer dapat mengetahui alur pembuatan kode pemgrograman tersebut.
